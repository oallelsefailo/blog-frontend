# MERN Blog App

This is a simple MERN app for creating and managing blog posts.

## Features

- Create, read, update, and delete blog posts.

## Technologies Used

- **Frontend**:

  - React
  - HTML/CSS
  - JavaScript

- **Backend**:
  - Node.js
  - Express.js
  - MongoDB (for database)

## Usage

- Visit the application in your web browser.
